Presentación de ECOruta

El sistema ECOruta permite coordinar y administrar de forma eficiente las rutas de recolección pública.

La plataforma facilita el control operativo mediante módulos organizados y conectados directamente a una base de datos.

Objetivo del sistema

Implementar una interfaz web intuitiva para administrar rutas, unidades, horarios y operadores de forma segura y eficiente.
